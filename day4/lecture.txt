перед структурой ставим & - выделяем память для объекта
& возвращает физический адрес объекта.
если не хотим работать с копиями, то используем &

Наследование реализовано в виде агрегации
НАследование - это когда класс является частным случаем другого класса (оно есть другой)
Агрегация - одно состоит из другого

New Make - экспортируемые (с большой буквы)
Make для создания структуры
Если пакет уже содержит имя, то не нужно делать функцию MakeAddress
достаточно сделать функцию make

Конвенция
New - возвращает адрес на объект
Make - возвращает копию объекта

по умолчанию сравниваются поля при сравнении структур
